# Автопарк — Консольное приложение (Вариант 1)

Консольное C-приложение для управления автопарком грузовых перевозок.

## Структура проекта
- `src/` — исходный код
- `includes/` — заголовочные файлы
- `db/` — SQL-скрипты и база данных
- `tests/` — unit-тесты
- `.github/workflows/` — CI/CD (GitHub Actions)

## Сборка и запуск

```bash
make          # обычная сборка
make check    # запуск тестов
make distcheck
./bin/autopark
