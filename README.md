# Project Name
Автопарк

# Description
Консольное приложение на C++ для учета машин, водителей и выполненных заказов автопарка. Проект переделан под предоставленную БД `CarPark.sql`: используются таблицы `Users`, `Drivers`, `Vehicles`, `Orders`, `Driver_earnings`. Система поддерживает аутентификацию и авторизацию: администратор управляет справочниками и заказами, водитель просматривает только разрешенные сведения. Водитель получает 20% от стоимости перевозки, а при добавлении заказа триггер запрещает груз тяжелее грузоподъемности машины.

# Installation
```bash
sudo apt install g++ make sqlite3 libsqlite3-dev
make
```

# Usage
```bash

```

# Database
Исходная схема лежит в `data/CarPark.sql`. Приложение подключается к готовой базе `data/carpark.db`.

Важно: при запуске приложение не добавляет демонстрационные данные автоматически. Сначала создайте/заполните базу самостоятельно, например:

```bash
sqlite3 data/carpark.db < data/CarPark.sql
sqlite3 data/carpark.db < data/seed.sql
```

Если `data/carpark.db` уже подготовлена в DB Browser for SQLite, просто оставьте ее в каталоге `data`.

Основные таблицы:
- `Users` — учетные записи пользователей;
- `Drivers` — водители автопарка;
- `Vehicles` — автомобили;
- `Orders` — выполненные заказы;
- `Driver_earnings` — начисления водителям за период.
